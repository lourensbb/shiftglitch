## How to Use This Document

All `{PLACEHOLDERS}` must be filled before handing this to an affiliate or embedding in the portal. The `{REFERRAL_URL}` and `{PROMO_CODE}` blocks are injected dynamically by the affiliate portal's JavaScript — do not pre-fill them.

In the affiliate portal, these templates are displayed with the affiliate's actual URL and promo code already substituted in via JavaScript.

## Video Scripts (TikTok / YouTube Shorts / Reels — 30 seconds)

### Script A — "The honest review"
```

[On camera, direct eye contact]
"I've been using {APP_NAME} for {DURATION} and here's what no one tells you about it."

PROBLEM (3–10s):
"If you're trying to {DESIRED_OUTCOME} and you're still doing {OLD_METHOD}, you're making it way harder than it needs to be."

REVEAL (10–22s):
[Screen record or app footage]
"This is {APP_NAME}. It {CORE_FEATURE_DEMO}. What used to take me {OLD_TIME_ESTIMATE} now takes {NEW_TIME_ESTIMATE}."

CTA (22–30s):
[Back on camera]
"My affiliate link gives you 10% off your first purchase. It's in my bio — or use promo code {PROMO_CODE} at checkout. Try it and tell me what you think."
```

### Script B — "The unexpected result"
```
HOOK (0–3s):

PROBLEM (3–8s):
"I've been stuck on {PROBLEM} for {TIME_PERIOD}. I'd tried {ALTERNATIVES} — none of it worked for me."

[App demo]
"With {APP_NAME}: {SPECIFIC_OUTCOME_1}. And {SPECIFIC_OUTCOME_2}. The thing that got me was {SURPRISING_DETAIL}."

"Link in bio, 10% off with my referral. Code is {PROMO_CODE}. This one's worth the look."
```

### Script C — "The numbers"

## Fill-In Reference for Brand Adaption

| Placeholder | Description |
|---|---|
| `{CORE_BENEFIT}` | The main thing users get from your app |
| `{PAIN_POINT}` | What they struggle with before using your app |
| `{TARGET_AUDIENCE}` | Who should see these captions |
| `{USE_CASE_1/2/3}` | Three specific things affiliates use the app for |
| `{COMMUNITY/NICHE}` | The community where this content will appear |
| `{APP_HASHTAG}` | Your app's hashtag (no `#` — added in template) |
| `{DURATION}` | e.g., "3 months", "6 weeks" — let affiliates personalise |
| `{STAT}` | Any measurable result (e.g., "47 sales", "3 months of content") |
| `{BEFORE_STATE}` | What life looked like before the app |
| `{AFTER_STATE}` | What improved after using the app |
| `{SURPRISING_DETAIL}` | The unexpected thing that impressed the affiliate |
| `{INITIAL_REASON}` | Why they tried the app in the first place |
